## 🚀 **Quick Start**

<div align="center">

![Installation Guide](.github/assets/installation.svg)

</div>

### 📦 **One-Command Installation**
```bash
# 🚀 Quick Install (Recommended)
curl -fsSL https://raw.githubusercontent.com/GeneticxCln/HyprSupreme-Builder/main/install.sh | bash

# 🆕 Enhanced Installation (New!)
./install_enhanced.sh --preset gaming --theme catppuccin

# 🧪 Start API Testing Server
npm start

# 🌐 Launch Community Platform
./launch_web.sh
```

### 🔧 **Manual Installation**
```bash
# 📥 Clone the consolidated repository
git clone https://github.com/GeneticxCln/HyprSupreme-Builder.git
cd HyprSupreme-Builder

# 🚀 Run the installer
chmod +x install.sh
./install.sh

# 🎯 Launch the application
./hyprsupreme

# 🧪 Run API integration tests
npm test
```

### 🐳 **Docker Installation**
```bash
# 📦 Run with Docker
docker-compose up -d

# 🌐 Access web interface
open http://localhost:5000

# 🧪 Access API testing suite
open http://localhost:3000
```

## 🛠️ **Requirements**

### 📋 **System Requirements**
- **OS**: Linux (Arch, Ubuntu 22.04+, Fedora 37+)
- **Desktop**: Wayland-compatible
- **RAM**: 4GB minimum, 8GB recommended
- **Storage**: 3GB free space (increased for API suite)
- **CPU**: Dual-core minimum, Quad-core recommended

### 📦 **Dependencies**
- Python 3.9+, Node.js 14+, Java 11+
- Hyprland 0.35+, Git, Curl, Maven
- Waybar, Rofi, Warp Terminal
- NetworkManager, PipeWire
