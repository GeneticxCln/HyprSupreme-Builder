## 🚀 Installation

### Quick Start
```bash
git clone https://github.com/GeneticxCln/HyprSupreme-Builder.git
cd HyprSupreme-Builder
chmod +x install.sh
./install.sh
```

### Unattended Installation
```bash
./install.sh --unattended --preset gaming
```

### Docker Installation
```bash
docker-compose up -d
```
